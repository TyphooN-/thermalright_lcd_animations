## Usage Examples

```bash
# Auto-rotate through ALL 55 animations (10 seconds each)
python src/animations.py

# Run specific animation
python src/animations.py --animation aurora

# Faster rotation (5 seconds per animation)
python src/animations.py --duration 5

# Faster frame rate
python src/animations.py --interval 0.03
```
